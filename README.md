# Global Timers

World Clocks - Stopwatch - Timer

A clean, minimal and production-ready time utility web application built with pure HTML, CSS and Vanilla JavaScript. No frameworks, no dependencies, no build tools required.

---

## Features

- Main clock displaying the current local time, date, day of week and day of year
- Six configurable world clocks with live updates using the Intl API
- High-precision stopwatch with start, pause, resume and reset controls
- Countdown timer with custom minutes and seconds, audible alarm on completion
- Animated canvas particle background with adjustable settings
- Full settings panel persisted via LocalStorage
- Progressive Web App support with offline functionality
- Installable on desktop and mobile devices
- Responsive layout for mobile, tablet and desktop
- No external libraries or CDN dependencies
